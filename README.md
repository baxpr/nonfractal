# nonfractal: MATLAB Toolbox for estimating nonfractal connectivity 

## Description 
It is a MATLAB toolbox for estimating nonfractal connectivity from a set of time series such as fMRI BOLD signals.

## Depends 
* Statistics Toolbox
* wmtsa

1.1
